# Air Dash

## 项目简介

**Air Dash** 是一款基于 [Godot 游戏引擎](https://godotengine.org/) 制作的 2D 横版平台跳跃类游戏。在游戏中，玩家将操控主角在精美的场景中跳跃、移动，并利用特殊能力与敌人和物体互动。除了传统的移动和跳跃能力，主角还可以通过向指定方向冲刺的方式穿越障碍，将敌人和物体向反方向抛出，从而解锁新的游戏机制和挑战。

本项目受到了《Ori》和其他同类型平台游戏的启发，致力于提供流畅的游戏体验以及富有创意的关卡设计。

## 特性

- **独特移动机制**：结合传统移动与跳跃，玩家可以通过冲刺与抛掷敌人和物体来解决难题或清除障碍。
- **精美的艺术风格**：采用独特的视觉风格，提供丰富多彩和细腻的游戏场景。
- **引人入胜的音效**：独特的音效设计为玩家带来沉浸式的游戏体验。
- **挑战性关卡**：多样化的关卡设计，考验玩家的反应能力和策略思考。
- **多人合作模式**（计划中）：未来将会加入多人合作功能，让玩家与好友一起畅玩。

## 安装与运行

1. 确保您的系统上已安装 [Godot 引擎](https://godotengine.org/download)。
2. 下载本项目源码，解压到您的计算机上。
3. 打开 Godot 引擎，选择 "Import Project"，然后找到解压后的项目文件夹。
4. 点击 "Import"，加载项目。
5. 直接点击 "Play" 按钮运行游戏。

## 控制

- **移动**：左右箭头键或 A/D 键
- **跳跃**：空格键
- **冲刺**：W 键（当接触敌人或物体时按下）
- **抛出**：按住冲刺键并按下方向键（朝向敌人或物体的反方向）
